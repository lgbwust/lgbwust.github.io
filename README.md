### 关于

基于Jekyll的静态博客网站，持续更新中.....

### 功能

> 首页
> 项目主页
> 在线pdf简历
> 博客
> 豆瓣读书(由于豆瓣api已经不再向个人开放，故此功能停用)
> 时间线

### 用途

- 写一些个人日记
- 记录一些工作中所遇到的问题及解决办法
- 随笔
- 完全属于个人的空间，不会有太多认识的人
- 放飞自我
- 给多年后的自己留一下一点回忆
- 以后想到了再添加

### 说明

- Github Pages自带Jekyll的编译功能，我们只需要上传md文档就能自动生成html静态页面。当需要本地预览时才需要安装本地编译环境
- 免费二级域名，无流量及空间限制
