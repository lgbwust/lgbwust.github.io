windows下tomcat端口被占用的问题

```
netstat -nao|findstr  8080 //显示端口占用情况
tasklist|findstr 8552 //查看占用的进程
tskill pid //结束端口占用
```

手机版chrome添加自定义搜索引擎
```
http://cn.bing.com/search?q=%s //地址栏输入该网址
```
然后就是settings => search engine => recent visits => 点击选择就可以了
